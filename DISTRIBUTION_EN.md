# Windows Distribution and Protection Guide (SmartScreen)

When distributing a `.exe` file generated by PyInstaller, Windows may display a "Windows protected your PC" warning (SmartScreen). This happens because the executable doesn't have a digital signature from a known developer.

Here are the ways to resolve or work around this:

## 1. Digital Signature (Professional Solution)
The only way to permanently remove the warning for all users is to sign the file with a **Code Signing Certificate**.
- **Cost:** Usually paid annually (e.g., DigiCert, Sectigo).
- **How it works:** You use a tool like `signtool.exe` from the Windows SDK to apply your signature to the `.exe`.

## 2. Use Installers (Recommended)
Instead of distributing a single `.exe`, use an installer creator like **Inno Setup** or **NSIS**.
- Installers tend to be more trusted by security systems than standalone executable files.
- They also allow for creating shortcuts in the Start Menu and Desktop in a standardized way.

## 3. Reputation Building
SmartScreen is based on "Reputation".
- The more people download and run your program (by clicking "More info" -> "Run anyway"), the faster reputation is built.
- Eventually, Windows will stop showing the warning for that specific file.

## 4. User Instructions (Immediate Solution)
Until you have a certificate, the best way is to instruct the user:
1. Click **"More info"**.
2. Click the **"Run anyway"** button that appears.

## 5. Manual Unblock
If Windows blocks execution completely:
1. Right-click the `.exe` file.
2. Go to **Properties**.
3. On the "General" tab, check the **"Unblock"** box (next to Security) and click OK.
