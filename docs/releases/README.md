# Release Artifacts

This folder stores version-specific release support files:

- `RELEASE_BODY_vX.Y.Z.md`: release text used in GitHub Releases.
- `RELEASE_CHECKLIST_vX.Y.Z.md`: operational checklist for validating and publishing a release.

Current examples:

- `RELEASE_BODY_v1.3.0.md`
- `RELEASE_CHECKLIST_v1.3.0.md`
- `RELEASE_BODY_v1.4.0.md`
- `RELEASE_CHECKLIST_v1.4.0.md`

`RELEASE_NOTES.md` remains at the repository root as the main version history.
